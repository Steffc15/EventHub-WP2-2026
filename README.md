# EventHub

## Description
EventHub is a web application where users can browse events and reserve seats. 
Users can create accounts and manage reservations, while administrators can add and manage events.

## Technologies
- Spring Boot
- Spring Security
- MySQL / PostgreSQL
- Bootstrap
- Docker

## User roles
- Client
- Admin

## User Stories

1. As a visitor I want to create an account so that I can reserve events.

2. As a client I want to browse events and reserve seats so that I can attend events.

3. As an admin I want to add and manage events so that users can see available events.
